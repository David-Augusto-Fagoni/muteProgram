#muteProgram

Este programa é responsável por mutar automaticamente o Spotify quando um anúncio estiver sendo reproduzido, utilizando o auxílio de um programa externo chamado SoundVolumeView.

Na primeira execução do jar, o programa criará automaticamente a pasta .spotify-muter no diretório do usuário junto do script powershell que será ultilizado.
Após isso, é necessário descompactar o arquivo do SoundVolumeView dentro dessa pasta.

Feito isso, o programa funcionará normalmente.
